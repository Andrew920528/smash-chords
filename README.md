# Git Tutorial

1. 在github create new branch

2. 開vs code 寫code
   git checkout branch-name <- go to the branch you just created
   git pull <- 雲端載下來
   git add . <- 我現在寫好code了 我要把它打包
   git commit -m "commit message" <- 我幫打包好的code加一個名字 別人才知道我在幹麻
   git push <- 把打包好的東西丟上雲端

3. 在github 開pull request
4. 把url 放在dc pr channel, 請別人檢查
